We consider the system UwqCG.

  Alphabet:

    O : [] --> a 
    lim : [a -> a] --> a 
    n : [] --> a 
    rec : [b * a -> b -> b * (a -> a) -> b -> b * a] --> b 
    rectuv : [b * a -> b -> b * (a -> a) -> b -> b * a] --> b 
    s : [a] --> a 

  Rules:

    rec(X, F, G, O) => X 
    rec(X, F, G, s(Y)) => F Y rec(X, F, G, Y) 
    rec(X, F, G, lim(H)) => G H rectuv(X, F, G, H n) 
    rectuv(X, F, G, n) => rec(X, F, G, n)
