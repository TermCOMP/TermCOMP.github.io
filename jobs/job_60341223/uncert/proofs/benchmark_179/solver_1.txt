We consider the system yg3FI.

  Alphabet:

    !280 : [N -> N * N] --> N 
    O : [] --> N 
    rec : [N -> (N -> N) -> N -> N * N] --> N -> N 
    s : [] --> N -> N 

  Rules:

    rec(F, !280(G, O)) => G 
    rec(F, !280(G, !280(s, X))) => F X rec(F, !280(G, X)) 
    !280(F, X) => F X 

This system is non-terminating, as demonstrated by the following reduction:

  rec(f, !280(/\x.rec(f, x) x, O)) !280(/\y.rec(f, y) y, O) 
    => (/\x.rec(f, x) x) !280(/\y.rec(f, y) y, O) 
    =>_beta rec(f, !280(/\x.rec(f, x) x, O)) !280(/\y.rec(f, y) y, O) 

That is, a term reduces to itself.
